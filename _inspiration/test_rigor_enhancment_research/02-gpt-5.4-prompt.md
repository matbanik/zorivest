### Research Topic 1: Designing E2E Test Architecture for Multi-Runtime Desktop Apps

   - Find blog posts or talks about testing local-first desktop apps with backend services
   - What subprocess management patterns exist (starting/stopping the Python server in test setup)?

2. **Test harness design for dual-process testing**:
   - The GUI process and the API process must both be running during E2E tests

   - Find implementations in open source projects

3. **Assertion strategies across process boundaries**:
