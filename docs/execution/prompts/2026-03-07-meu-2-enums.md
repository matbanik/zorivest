  - `ImageOwnerType`: `TRADE="trade"`, `REPORT="report"`, `PLAN="plan"`
  - `DisplayModeFlag`: `HIDE_DOLLARS="hide_dollars"`, `HIDE_PERCENTAGES="hide_percentages"`, `PERCENT_MODE="percent_mode"`
- AC-2: `enums.py` defines these expansion enums exactly:
  - `RoundTripStatus`: `OPEN="open"`, `CLOSED="closed"`, `PARTIAL="partial"`
  - `IdentifierType`: `CUSIP="cusip"`, `ISIN="isin"`, `SEDOL="sedol"`, `FIGI="figi"`
  - `FeeType`: `COMMISSION="commission"`, `EXCHANGE="exchange"`, `REGULATORY="regulatory"`, `ECN="ecn"`, `CLEARING="clearing"`, `PLATFORM="platform"`, `OTHER="other"`
  - `StrategyType`: `VERTICAL="vertical"`, `IRON_CONDOR="iron_condor"`, `BUTTERFLY="butterfly"`, `CALENDAR="calendar"`, `STRADDLE="straddle"`, `STRANGLE="strangle"`, `CUSTOM="custom"`

  - `RoutingType`: `PFOF="pfof"`, `DMA="dma"`, `HYBRID="hybrid"`
  - `TransactionCategory`: `DEPOSIT="deposit"`, `WITHDRAWAL="withdrawal"`, `TRANSFER_IN="transfer_in"`, `TRANSFER_OUT="transfer_out"`, `FEE="fee"`, `INTEREST="interest"`, `DIVIDEND="dividend"`, `ACH_DEBIT="ach_debit"`, `ACH_CREDIT="ach_credit"`, `WIRE_IN="wire_in"`, `WIRE_OUT="wire_out"`, `CHECK="check"`, `CARD_PURCHASE="card_purchase"`, `REFUND="refund"`, `ATM="atm"`, `OTHER="other"`

- AC-3: Every enum class subclasses `StrEnum`, not `Enum`, and every member value matches the build plan exactly, including case and underscores.
- AC-4: The module contains exactly the 14 enum classes explicitly listed in `01-domain-layer.md` section 1.2. Do not invent a missing 15th class.

- Must not "correct" enum values using other docs when they conflict with `01-domain-layer.md` section 1.2.
- Must not broaden scope beyond `enums.py`, `test_enums.py`, and minimal import wiring if needed.
