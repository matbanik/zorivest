## Design Decisions & Known Risks

- **Decision**: `BrokerPort` methods return `dict` / `list[dict]` — **Reasoning**: Build plan §1.5 specifies these as untyped dicts. Concrete DTOs will come in MEU-6. `# type: ignore[type-arg]` comments suppress pyright warnings on bare `dict`. — **ADR**: inline (minor)
- **Decision**: `UnitOfWork.__exit__` uses `*args: object` — **Reasoning**: Protocol signature doesn't need the full `(exc_type, exc_val, tb)` tuple — consumers care about the existence of the method, not the exact parameter shape.
- **Risk**: `BrokerPort` and `BankImportPort` return untyped dicts — will need refinement when concrete adapters are built (Phase 2)
