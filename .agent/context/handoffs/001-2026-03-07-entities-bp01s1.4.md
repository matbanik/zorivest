## Design Decisions & Known Risks

- **Decision**: `report` field typed as `Optional[object]` — **Reasoning**: TradeReport is P1, not in scope. Using `object` avoids forward-ref complexity. — **ADR**: inline (minor)
- **Decision**: `ImageAttachment.__post_init__` rejects non-webp mime — **Reasoning**: domain-model-ref line 107 says "always image/webp". Frozen dataclass requires validation at construction. — **ADR**: inline (critical)
- **Decision**: Trade and Account are mutable (`@dataclass`) — **Reasoning**: Trade needs `attach_image()`, Account needs mutable balance_snapshots list.
- **Assumption**: `images: list[ImageAttachment] = field(default_factory=list)` uses factory to avoid mutable default sharing
- **Risk**: `report: Optional[object]` will need re-typing when TradeReport is implemented in a future MEU
