- **Decision**: `followed_plan` UI is a checkbox toggle (not a select) â€” **Reasoning**: API accepts `bool`; a binary input maps more naturally than a 3-state dropdown. **ADR**: inline.
- **Source Basis**: `docs/build-plan/06b-gui-trades.md Â§20`; API schema in `packages/api/src/zorivest_api/routes/reports.py`
- **Risk**: Gradeâ†’star reverse mapping assumes consistent letter-grade values from API. If API sends unexpected grades, the star defaults to 0.
